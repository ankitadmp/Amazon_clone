# Amazon_clone
In this the html and css is used to create the amazon clone,it's my first project
A responsive Amazon-inspired e-commerce homepage built using HTML5 and CSS3. This project recreates the look and feel of Amazon's user interface while focusing on clean code, responsive design, and modern front-end development practices.

Features:
Responsive navigation bar
Search bar UI
Hero section/banner
Product category cards
Product showcase sections
Responsive layout using Flexbox and CSS Grid
Footer with multiple sections
Clean and organized code structure
Technologies Used
HTML5
CSS3
Flexbox
CSS Grid
